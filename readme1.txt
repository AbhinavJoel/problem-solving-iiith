How to fork a repository?
1.Open the link -  https://github.com/virtual-labs/problem-solving-iiith
2.Click code and copy the url given below
3.Open terminal and navigate to the directory which we are working on
4.Type git clone and the url we copied earlier and press enter
5.Type cd problem-solving-iiith
6.Type git remote -v and press enter
7.Type git remote add upstream, and then paste the URL you copied above and press enter 
8.To verify the new upstream repository we've specified for our fork, type git remote -v again. We should see the URL for our fork as origin, and the URL for the original repository as upstream.
9.Create a new file called readme1.txt
10.Continue to add commit and push.
